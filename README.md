[Beadcraft-Studio]: The Ultimate AI-Powered Fuse Bead Blueprint Generator

[Beadcraft-Studio] is a modern, beautifully designed web application that bridges the gap between digital images and physical crafting. It seamlessly transforms everyday photos into detailed, ready-to-build Fuse Bead (Perler Bead) blueprints.

Built with a strong focus on tactile UI/UX and DIY practicality, this tool solves the common problem of messy image downscaling by offering an intelligent, multi-step processing pipeline.

✨ Core Features:

AI-Enhanced Pixelation: Integrates advanced AI vision models to restyle complex photos into clean, high-contrast 8-bit pixel art before conversion, preserving hard edges and reducing visual noise.

Authentic Color Mapping: Strictly maps digital pixels to a built-in database of 220+ real-world fuse bead colors. It uses precise color distance algorithms to ensure what you see on screen is exactly what you can build with physical beads.

Custom Pegboard Sizing: Flexible resolution settings (e.g., 30x30, 50x50) that perfectly align with standard physical pegboard dimensions.

Interactive Blueprints & BOM: Automatically generates a grid overlay with hoverable color codes and a comprehensive Bill of Materials (BOM), tallying the exact number of beads needed for each specific color.

Tactile User Interface: A meticulously crafted, responsive frontend featuring Polaroid-inspired layouts, smooth toggle interactions, and a strict, shift-free layout system.

🚀 Roadmap:
Starting as a robust frontend MVP, the project is actively evolving into a full-stack application. Upcoming features include Supabase integration for user authentication and cloud galleries, alongside a Stripe-powered payment system for premium high-resolution exports.
